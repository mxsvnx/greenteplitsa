# Тепличное хозяйство — вариант 4

Проект про управление работами в тепличном хозяйстве агрокомплекса.

Есть теплицы (каждая под тип выращиваемой продукции) и овощные культуры. Для каждой культуры задаётся собственный график работ: посев, полив, внесение удобрений и сбор урожая. Далее формируется цикл выращивания (теплица + культура + период), и по графику культуры автоматически создаётся план работ на этот цикл.

После выполнения работ в цикл вносится “факт”: дата выполнения и затраченные ресурсы (вода, удобрения, трудозатраты), а также объём полученного урожая. По фактическим работам считаются итоги по циклу (суммарно по воде/удобрениям/труду/урожаю).

## Реализовано

- Справочник теплиц.
- Справочник овощных культур.
- График работ по каждой культуре (посев/полив/удобрения/сбор).
- Циклы выращивания (привязка культуры к теплице на период).
- Генерация плана работ по циклу на основе графика культуры.
- Внесение фактических выполненных работ с ресурсами и урожаем.
- Итоги по циклу (суммарные ресурсы и урожай).

## Роли

Редактирование данных, генерация плана и внесение факта доступны пользователям с правами главного агронома (группа `chief_agronomist`) и суперпользователю. Просмотр доступен авторизованным пользователям.
