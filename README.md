# Система управления тепличным хозяйством

Веб-приложение для комплексного управления операциями в теплицах агрокомплекса. Позволяет отслеживать выращивание культур, планировать сельскохозяйственные работы и анализировать расход ресурсов.

## Описание функционала

Система базируется на следующих сущностях:

- *Основные возможности

- Создание и управление справочниками (теплицы, культуры, графики)
- Автоматическое планирование сельскохозяйственных работ
- Фиксация фактических работ с указанием затраченных ресурсов (вода, удобрения, время)
- Отслеживание урожайности по циклам
- Автоматический расчёт общих показателей по циклу (суммарные ресурсы и выход продукции)
- Справочник теплиц.
- Справочник овощных культур.
- График работ по каждой культуре (посев/полив/удобрения/сбор).
- Циклы выращивания (привязка культуры к теплице на период).
- Генерация плана работ по циклу на основе графика культуры.
- Внесение фактических выполненных работ с ресурсами и урожаем.
- Итоги по циклу (суммарные ресурсы и урожай).

## Роли

РедПрава доступа

- **Просмотр** - доступен всем авторизованным пользователям
- **Редактирование, планирование, учёт** - только главный агроном (группа `chief_agronomist`) и администратор

## Требования

- Python 3.8+
- Django 5.0+
- SQLite3

## Установка и запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Применить миграции базы данных
python manage.py migrate

# Создать администратора
python manage.py createsuperuser

# (Опционально) Загрузить начальные данные
python manage.py seed_greenhouse

# Запустить сервер разработки
python manage.py runserver
```

Приложение будет доступно по адресу `http://127.0.0.1:8000/`

Администраторская панель: `http://127.0.0.1:8000/admin/`
